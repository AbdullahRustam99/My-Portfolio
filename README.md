# Abdullah Rustam — Portfolio

Personal portfolio built with Next.js showcasing projects, skills, and services as an Agentic AI Engineer & Full-Stack Developer.

## Tech Stack

- **Framework:** Next.js 15 (App Router, Turbopack)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4, `tw-animate-css`
- **Animation:** Framer Motion, Motion
- **Particles:** tsParticles (image particles with tech icons)
- **UI Components:** Radix UI Slot, Lucide icons
- **Fonts:** Fira Code, Orbitron (via next/font)

## Sections

| Section     | Description                                |
| ----------- | ------------------------------------------ |
| Hero        | Typewriter greeting, role rotation, CTA    |
| About       | Bio and background                         |
| Services    | Service offerings                          |
| Skills      | Tech categories with skill cards           |
| Portfolio   | Project cards with image fallback          |
| Contact     | Contact form with social links             |
| Footer      | Site footer                                |

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Lint

```bash
npm run lint
```

## Project Structure

```
src/
├── app/          # Layout, pages, globals.css
├── components/
│   ├── sections/ # Hero, About, Services, Skills, Portfolio, Contact, Footer, Navbar
│   └── ui/       # Reusable components (button, card, particles, preloader, etc.)
├── data/         # projects.json
└── lib/          # Utilities, theme toggle
```
